### Products

🔵 **Aegis Reliability** — Predictive maintenance. ML-based vibration analysis, health scoring, early warning, incident management with response playbooks.

🔴 **Aegis Perimeter** — Physical security. YOLOv8 computer vision, zone-based threat scoring, breach detection, automated response workflow.

🟢 **Aegis Portal** — Unified Mission Control. Single pane of glass across all products. JWT auth, entity registry, real-time event bus.

### Get Started
```bash
git clone https://github.com/aegis-platform/aegis.git
cd aegis && ./launch.sh
# → Portal:      http://localhost:3000 (admin/admin)
# → Reliability:  http://localhost:8000
# → Perimeter:    http://localhost:8001
```
