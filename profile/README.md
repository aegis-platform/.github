## What We Build

- **Mission Control** for operators, sites, assets, incidents, and autonomous systems
- **Security Systems** for site protection, threat scoring, and response workflows
- **Technical Resilience** for predictive maintenance, health scoring, and uptime intelligence
- **Edge AI Infrastructure** for on-prem, low-latency, real-world deployment
- **Applied Autonomy** for human-in-the-loop control of complex machines and environments

## Live Programs

- **Aegis Reliability**  
  Condition intelligence for industrial equipment: vibration analysis, health scoring, early warning, incidents, and maintenance playbooks.

- **Aegis Perimeter**  
  Physical security for complex sites: computer vision, zone-based threat scoring, breach detection, and response workflows.

- **Mission Control**  
  Unified command layer across programs: auth, entity registry, real-time event bus, and operational overview.

## Platform Thesis

We do not build isolated dashboards or one-off AI demos.

We build **operational systems**:

**sense -> interpret -> prioritize -> respond -> record**

One shared engineering backbone.
Multiple domain-specific programs on top.

## Get Started

```bash
git clone https://github.com/aegis-platform/aegis.git
cd aegis
./launch.sh

# Company site:    http://localhost:3000
# Mission Control: http://localhost:3000/mission-control  (admin/admin)
# Reliability:     http://localhost:8000
# Perimeter:       http://localhost:8001
```
